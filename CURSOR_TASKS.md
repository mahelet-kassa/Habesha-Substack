1. Scaffold the React + TypeScript app and install dependencies.
2. Build core layout (Header, Footer, App shell, routing).
3. Implement bilingual language state and theme toggle.
4. Create core pages: Home, Post, Editor, Profile, Subscribe, Search, Analytics.
5. Build reusable UI components (PostCard, TagList, ReactionBar, CommentThread).
6. Add editor UX: toolbar, bilingual side-by-side fields, scheduling + visibility options.
7. Add subscription plans & checkout flow placeholders.
8. Implement comments, reactions, and moderation UI scaffolding.
9. Add analytics dashboards and growth insights.
10. Add search filters (author, topic, language) and featured feed sections.
11. Implement SEO helpers (title, meta tags, canonical URL).
12. Add accessibility checks (ARIA labels, keyboard focus states).
13. Wire to API contracts with React Query or equivalent.
14. Add content export options (PDF/Markdown) placeholders.
15. Prepare production build and deploy pipeline.
